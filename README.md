# infdevtp
-----------------------
-----------------------
A little collection of teleport jarmods i made because retrowrapper doesnt work properly for these versions. Sadly i cant publish source code because it contains mojang stuffs, though you can always just decompile and check it urself. Idk, im not a lawyer...
-----------------------
-----------------------
How to use:

1. Install prism launcher/multiMC
2. Install the needed version
3. Click "add to minecraft jar" and choose the downloaded zip for the respected version
4. Choose java 8, and run the game.
-----------------------
-----------------------
Features:
By entering the pause screen youll find a new button (or press f12 while in game)
It opens the GUI where you input the X, Y, Z coordinates. If you select the "Place on ground" option Y coordinate will be ignored and you will be placed on the first non-air block counting from the sky.
Also a small overlay displaying X,Y,Z coordinates.
